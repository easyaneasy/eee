# 블록 요소
마크다운 문서를 구조화한다.

## 헤더(h)
#을 사용하여 표현하고. 1부터 6까지 있는데 숫자가 커질수록 글자 크기가 작아진다.

## 리스트(list)
-(dash)를 사용해서 표현하고, 항목을 나열한다.

## 넘버링리스트
1., 2., 3., 을 사용해서 표현하고, 순서가 있는 항목을 나열한다.

## 코드 블록
`(백틱) 위 아래 3개를 사용해서 표현하고, 코드를 표현한다.

## 구분선 (Horizontal rule)
-(dash) 3개를 사용해서 표현하고, 섹션을 구분한다.
